# مستشفى ابن الحاج التخصصي - Al Mahwit

موقع ويب تعريفي لمستشفى ابن الحاج التخصصي الواقع في المحويت - همدان - إبلاس.

## المميزات:
* **تأسس عام 2026**.
* يعمل على مدار **24 ساعة**.
* تصميم متجاوب (Responsive) يعمل على الجوال والكمبيوتر.
* نظام حجز مباشر عبر الواتساب.
* واجهة باللغتين العربية والإنجليزية.
